Installation Instructions

After downloading the project files:

  1. Extract the ZIP file to your preferred location
  2. Navigate to the project directory and run npm install
  3. Configure your environment variables in a .env file based on the provided .env.example
  4. Set up a PostgreSQL database and update the connection string in your .env file
  5. Run npm run db:push to set up the database schema
  6. Start the application with npm run dev (development) or npm run start (production)

For detailed deployment instructions, refer to the documentation files included in the package.
